# KL annealing experiment

Contents:

- `simulate_kl_annealing.py`: the runnable scalar-VAE simulation used in the article.
- `requirements.txt`: the dependency versions used to verify the experiment.
- [`final_metrics.csv`](../../data/kl-annealing/final_metrics.csv): final results from the experiment.

The complete code is also embedded in the article, [Why KL Annealing Helps VAEs](../../../posts/why-kl-annealing-helps-vaes/).

Run the code with:

```bash
python -m pip install -r requirements.txt
python simulate_kl_annealing.py
```
